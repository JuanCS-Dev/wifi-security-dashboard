### Game Won't Start

**Problem:** Double-clicking doesn't launch the game

**Solutions:**

**Windows:**
- Right-click â†’ "Run as administrator"
- Check if antivirus blocked it (add exception)
- Reinstall the game

**macOS:**

- System Preferences â†’ Security â†’ "Open Anyway"

- Check if FUSE is installed: `sudo apt install fuse`
- Try .deb package instead

---

### Game Runs Slowly

**Problem:** FPS below 30, game lags

**Solutions:**
- Close other programs
- Reduce screen resolution
- Try fullscreen mode (F11)
- Check system requirements:
  - Python 3.11+
  - 4GB RAM minimum
  - Modern CPU (2015+)

---

### Can't See Characters/Text

**Problem:** Emojis show as boxes or are missing

**Solutions:**
- Update your operating system
- Install emoji fonts:
  - Windows: Should work by default
  - macOS: Should work by default
  - Linux: `sudo apt install fonts-noto-color-emoji`
- Restart the game

---

**Problem:** Can't read messages in time

**Solutions:**
- Press **P** to pause
- Read at your own pace
- Dialog timing: 3-5 seconds per message
- Restart scenario if needed (press 1/2/3)
